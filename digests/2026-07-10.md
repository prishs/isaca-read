# Security & Audit Digest — 2026-07-10

## Top items

### 1. CISA adds 4 actively exploited flaws to KEV catalog — federal remediation due today
**Why it matters for audit/CISO/VAPT:** All four carry a 2026-07-10 federal remediation deadline, a live benchmark for testing patch-SLA compliance and KEV-tracking controls.
CISA added Adobe ColdFusion path traversal (CVE-2026-48282, CVSS 10.0), Langflow authorization bypass (CVE-2026-55255), and page-builder flaws in JoomShaper SP (CVE-2026-48908) and Joomlack (CVE-2026-56290) to its Known Exploited Vulnerabilities catalog, all under confirmed active exploitation. ([The Hacker News](https://thehackernews.com/2026/07/cisa-adds-4-actively-exploited-adobe.html))

### 2. npm 12 disables install scripts by default to curb supply-chain risk
**Why it matters for audit/CISO/VAPT:** A default-security change in the world's largest package registry directly affects dependency-management and build-pipeline controls that audits test against.
GitHub released npm v12 with install scripts off by default and deprecated granular access tokens that could bypass 2FA, closing a long-standing post-install compromise vector. ([The Hacker News](https://thehackernews.com/2026/07/npm-12-disables-install-scripts-by.html))

### 3. China-linked hackers exploit Roundcube flaws to spy on university researchers
**Why it matters for audit/CISO/VAPT:** A textbook control-failure case — unpatched, publicly known webmail flaws left a nation-state actor inside research networks — relevant to patch-cadence and third-party risk audits.
A suspected China-aligned cluster exploited now-patched, critical Roundcube vulnerabilities at U.S. and Canadian universities to steal credentials and deploy backdoors; also covered on this week's Cyber Security Headlines (see below). ([BleepingComputer](https://www.bleepingcomputer.com/news/security/hackers-exploit-roundcube-flaw-to-spy-on-academic-researchers/))

### 4. Writer AI patches critical cross-tenant session-leak flaw in agent previews
**Why it matters for audit/CISO/VAPT:** A one-click bug that could expose one tenant's session data to another strikes at the core assurance enterprise GenAI platforms claim — a board-level trust issue for orgs piloting agentic AI.
Researchers disclosed "WriterSession," a now-patched vulnerability in the Writer enterprise AI platform that could have let agent preview links leak session tokens across customer tenants. ([The Hacker News](https://thehackernews.com/2026/07/writer-ai-flaw-could-let-agent-previews.html))

### 5. GhostApproval: symlink flaws could let malicious repos hijack six AI coding agents
**Why it matters for audit/CISO/VAPT:** Shows AI coding assistants that approve file edits are a new sandbox-escape surface — developer-tooling risk that VAPT scopes should now include.
Wiz researchers found symlink-traversal flaws in six popular AI coding assistants (including Claude Code, Cursor, and Codex) that let a booby-trapped repository redirect an approved file edit onto a sensitive file instead. ([The Hacker News](https://thehackernews.com/2026/07/ghostapproval-symlink-flaws-could-let.html))

### 6. "Friendly Fire": AI agents built to catch malicious code can be tricked into running it
**Why it matters for audit/CISO/VAPT:** Undermines the assumption that AI-assisted code review is a safe, non-executing control — a direct hit to audit programs relying on AI scanners.
The AI Now Institute published a proof-of-concept showing AI coding agents tasked with scanning open-source repos for vulnerabilities can instead be socially engineered into executing the attacker's payload. ([The Hacker News](https://thehackernews.com/2026/07/friendly-fire-ai-agents-built-to-catch.html))

### 7. Microsoft patches RoguePlanet Defender flaw granting SYSTEM privileges
**Why it matters for audit/CISO/VAPT:** A privilege-escalation bug in the defense tool itself, patched roughly a month after public disclosure — a concrete patch-SLA data point for endpoint-security governance.
Microsoft shipped a fix for RoguePlanet (CVE-2026-50656, CVSS 7.8), a privilege-escalation flaw in the Malware Protection Engine; also covered on this week's SANS Stormcast (see below). ([The Hacker News](https://thehackernews.com/2026/07/microsoft-patches-rogueplanet-defender.html))

### 8. GodDamn ransomware uses PoisonX driver to blind endpoint defenses
**Why it matters for audit/CISO/VAPT:** Kernel-level EDR/XDR evasion via a driver is now a standard ransomware precursor step — a control gap VAPT engagements should specifically test for.
Symantec's Threat Hunter Team flagged GodDamn, a new ransomware family that loads the PoisonX kernel driver to disable endpoint security software before encrypting victims. ([The Hacker News](https://thehackernews.com/2026/07/goddamn-ransomware-uses-poisonx-driver.html))
