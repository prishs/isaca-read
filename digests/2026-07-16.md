# Security & Audit Digest — 2026-07-16

## Top items

### 1. Krebs: postmortem on CISA's own six-month GitHub credential leak
**Why it matters for audit/CISO/VAPT:** A textbook case for third-party/contractor access reviews and secrets-scanning gaps — even the agency that runs KEV had a contractor leak AWS GovCloud keys undetected for months.
CISA published a postmortem after a contractor published dozens of internal CISA credentials, including AWS GovCloud keys, in a public GitHub repo for nearly six months before Krebs notified them. ([source](https://krebsonsecurity.com/2026/07/lessons-learned-from-cisas-recent-github-leak/))

### 2. CISA adds Oracle E-Business Suite and KNX flaws to KEV
**Why it matters for audit/CISO/VAPT:** Two more hard remediation deadlines land this week — confirm asset inventories catch both ERP and OT/building-automation exposure.
CVE-2026-46817 (Oracle E-Business Suite privilege management, due 2026-07-18) and CVE-2023-4346 (KNX Protocol account-lockout bypass, due 2026-07-29) were added to CISA's KEV catalog on 2026-07-15 based on active exploitation. ([source](https://www.cisa.gov/news-events/alerts/2026/07/15/cisa-adds-two-known-exploited-vulnerabilities-catalog))

### 3. Zoom warns of critical account takeover flaw (CVSS 9.8)
**Why it matters for audit/CISO/VAPT:** Unauthenticated account takeover in a widely deployed collaboration client — patch verification should be prioritized ahead of the next KEV deadline cycle.
Zoom disclosed a critical vulnerability in its Windows desktop client and SDK that an unauthenticated attacker could exploit to hijack accounts. ([source](https://www.bleepingcomputer.com/news/security/zoom-warns-of-critical-account-takeover-vulnerability/))

### 4. Compromised AsyncAPI npm packages deliver botnet malware
**Why it matters for audit/CISO/VAPT:** Another open-source supply-chain compromise — audit should confirm lockfiles don't pin the affected versions and that dependency scanning covers transitive packages.
Four malicious versions of packages in the @asyncapi namespace (including @asyncapi/generator-helpers@1.1.1) delivered a multi-stage botnet loader, per OX Security, SafeDep, Socket, and StepSecurity. ([source](https://thehackernews.com/2026/07/compromised-asyncapi-npm-packages.html))

### 5. New public PoC for Windows privilege-escalation flaw, hours after Patch Tuesday
**Why it matters for audit/CISO/VAPT:** A working exploit ("LegacyHive") for a User Profile Service hive-load elevation bug is now public — VAPT teams should confirm this month's patches actually landed before it's weaponized further.
Researcher Chaotic Eclipse released a PoC for a Windows User Profile Service arbitrary hive-load elevation-of-privilege vulnerability shortly after this month's record Patch Tuesday. ([source](https://thehackernews.com/2026/07/researcher-drops-new-windows-zero-day.html))

### 6. Cursor editor flaw lets a cloned repo silently execute code on Windows
**Why it matters for audit/CISO/VAPT:** Developer tooling is an under-reviewed attack surface; opening an untrusted repo can trigger arbitrary execution with no prompt.
If a malicious repository opened in Cursor on Windows contains a file named `git.exe` in the project root, Cursor runs it automatically — no click or warning. ([source](https://thehackernews.com/2026/07/cursor-flaw-lets-malicious-cloned.html))

### 7. Microsoft: a year of ShinyHunters-style Salesforce breaches used no exploits at all
**Why it matters for audit/CISO/VAPT:** Attackers walked in via abused OAuth app trust and connected-app consent, not vulnerabilities — a governance gap standard vuln-management programs won't catch.
Microsoft mapped three attack paths, consistent with ShinyHunters activity over the past year, that abused legitimate Salesforce trust relationships (connected apps, OAuth tokens) rather than platform flaws. ([source](https://thehackernews.com/2026/07/microsoft-maps-year-long-shinyhunters.html))

### 8. Claude for Chrome flaw let rogue extensions trigger Gmail/Docs reads
**Why it matters for audit/CISO/VAPT:** As AI browser agents get inbox and document access, extension-isolation failures become a new data-exposure vector to add to endpoint/browser governance reviews.
Researchers found that any browser extension able to run script on claude.ai could trigger Claude for Chrome to read a user's Gmail, latest Google Doc, and Calendar. ([source](https://thehackernews.com/2026/07/claude-for-chrome-flaw-lets-other.html))

### 9. Firefox, Chrome, Adobe, VMware ship critical fixes; Firefox PoC already public
**Why it matters for audit/CISO/VAPT:** Two Firefox flaws have published exploit code — treat as urgent patch-compliance items this cycle, not routine browser updates.
Mozilla patched two critical Firefox flaws (CVE-2026-15718, CVE-2026-15719) with public exploit code, alongside critical updates from Chrome, Adobe, and VMware. ([source](https://thehackernews.com/2026/07/firefox-chrome-adobe-and-vmware-updates.html))

### 10. Microsoft blocks July updates on some Dell PCs after shutdown reports
**Why it matters for audit/CISO/VAPT:** A patch-compatibility failure mid-KEV-deadline crunch — confirm compensating controls for any Dell fleet stuck without this month's fixes.
Microsoft is withholding this month's Windows 11 security updates from some Dell devices after reports of shutdowns and performance issues tied to Intel's platform firmware framework. ([source](https://www.bleepingcomputer.com/news/microsoft/microsoft-some-dell-devices-shut-down-after-windows-update/))
