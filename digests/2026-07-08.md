# Security & Audit Digest — 2026-07-08

## Top items

### 1. Langflow authorization bypass added to CISA KEV
**Why it matters for audit/CISO/VAPT:** Federal remediation deadline is already tight (2026-07-10), and Langflow is increasingly embedded in AI-agent build pipelines — a live patch-compliance item auditors should flag now.
CVE-2026-55255 lets an attacker bypass authorization in Langflow via a user-controlled key. CISA added it to the Known Exploited Vulnerabilities catalog on July 7 based on confirmed exploitation. ([CISA](https://www.cisa.gov/news-events/alerts/2026/07/07/cisa-adds-three-known-exploited-vulnerabilities-catalog))

### 2. Adobe ColdFusion path traversal now exploited, added to KEV
**Why it matters for audit/CISO/VAPT:** Max-severity, actively exploited, federal deadline July 10 — a top-priority patch item for any org running ColdFusion.
CVE-2026-48282 is a path-traversal flaw in Adobe ColdFusion; KEVIntel confirmed in-the-wild exploitation and CISA added it to KEV on July 7. ([BleepingComputer](https://www.bleepingcomputer.com/news/security/max-severity-adobe-coldfusion-flaw-now-exploited-in-attacks/))

### 3. Accenture confirms breach, 35GB of source code and data stolen
**Why it matters for audit/CISO/VAPT:** Accenture sits deep in the third-party/audit supply chain for Fortune 500 clients — stolen source code and internal data is a cascading vendor-risk exposure.
A threat actor claims to have stolen 35GB of Accenture source code and other data and is offering it for sale; Accenture has confirmed the breach but not disclosed scope or ransom demands. ([BleepingComputer](https://www.bleepingcomputer.com/news/security/accenture-confirms-breach-after-hacker-offers-stolen-data-for-sale/))

### 4. Critical session-isolation flaw in Writer AI could leak tokens across tenants
**Why it matters for audit/CISO/VAPT:** GenAI SaaS platforms are being adopted for audit and compliance drafting; a cross-tenant break undermines the isolation guarantees those engagements assume.
The now-patched "WriteEscape" flaw in Writer, an enterprise GenAI platform, allowed a one-click attack to leak session tokens across tenants via agent preview functionality. ([The Hacker News](https://thehackernews.com/2026/07/writer-ai-flaw-could-let-agent-previews.html))

### 5. Public GitHub issue could trick Agentic Workflows into leaking private repos
**Why it matters for audit/CISO/VAPT:** No credentials required — a normal-looking public issue is enough to exfiltrate private repo contents, including policy-as-code and compliance configs stored in CI/CD.
Noma Security showed that GitHub Agentic Workflows can be manipulated via a crafted public issue to leak an organization's private repository data, with no stolen access needed. ([The Hacker News](https://thehackernews.com/2026/07/public-github-issue-could-trick-github.html))

### 6. Rogue-agent flaw could have let attackers hijack Google Dialogflow CX chatbots
**Why it matters for audit/CISO/VAPT:** Dialogflow bots often front helpdesk and vendor-intake workflows; project-wide agent compromise means one breached agent can pivot to read live conversations across an org's chatbots.
A now-fixed flaw let anyone with edit rights on one Code Block-enabled Dialogflow CX agent compromise other agents in the same GCP project, exposing live conversation data. ([The Hacker News](https://thehackernews.com/2026/07/rogue-agent-flaw-could-have-let.html))

### 7. China-aligned actors exploit Roundcube flaws against university mail servers
**Why it matters for audit/CISO/VAPT:** Illustrates patch-lag risk turning into targeted IP-theft campaigns — a useful control-failure case study for research-heavy or export-controlled environments.
A suspected China-aligned cluster exploited now-patched critical Roundcube webmail vulnerabilities against physics and engineering departments at U.S. and Canadian universities. ([The Hacker News](https://thehackernews.com/2026/07/suspected-china-aligned-hackers-exploit.html))

### 8. "Januscape" — 16-year-old Linux KVM flaw allows guest-to-host VM escape
**Why it matters for audit/CISO/VAPT:** Undermines the hypervisor isolation many audit and compliance labs rely on for safe test environments; long-lived kernel bugs like this complicate patch-cadence assurances.
CVE-2026-53359 is a use-after-free in KVM's shadow-MMU code, exploitable from a guest VM to corrupt host kernel state, affecting both Intel and AMD systems. ([The Hacker News](https://thehackernews.com/2026/07/16-year-old-linux-kvm-flaw-lets-guest.html))
