# Security & Audit Digest — 2026-07-22

## Top items

### 1. WordPress wp2shell RCE Chain Actively Exploited, Added to CISA KEV
**Why it matters for audit/CISO/VAPT:** A combined unauthenticated RCE chain is under mass exploitation with a federal remediation deadline of 2026-07-24 — any WordPress Core asset in scope needs immediate patch verification.
Public exploit code for CVE-2026-63030 and CVE-2026-60137 is fueling mass scanning and webshell deployment across unpatched sites; both were added to the CISA KEV catalog on 2026-07-21. ([source](https://thehackernews.com/2026/07/wordpress-wp2shell-exploitation-grows.html))

### 2. CISA Adds Langflow and DD-WRT Flaws to KEV Catalog
**Why it matters for audit/CISO/VAPT:** New KEV entries carry binding federal remediation deadlines and are a useful benchmark for testing an organization's own patch-SLA compliance.
On 2026-07-21 CISA added CVE-2026-0770 (Langflow) and CVE-2021-27137 (DD-WRT buffer overflow) to the KEV catalog alongside the WordPress CVEs above, each with evidence of active exploitation and a 2026-07-24 remediation deadline. ([source](https://www.cisa.gov/news-events/alerts/2026/07/21/cisa-adds-four-known-exploited-vulnerabilities-catalog))

### 3. SharePoint RCE (CVE-2026-50522) Exploited to Steal Machine Keys
**Why it matters for audit/CISO/VAPT:** Attackers are stealing machine keys to retain access even after patching — patch confirmation alone isn't sufficient remediation evidence for this one.
The critical deserialization flaw (CVSS 9.8), the third actively exploited SharePoint RCE from July's Patch Tuesday, is being used per watchTowr reporting to persist post-patch. ([source](https://thehackernews.com/2026/07/critical-sharepoint-rce-cve-2026-50522.html))

### 4. Qilin Ransomware Uses PAN-OS Auth Bypass for Initial Access
**Why it matters for audit/CISO/VAPT:** A ransomware group operationalizing a patched VPN vulnerability weeks after disclosure is a clear control-failure lesson on perimeter patch latency.
Arctic Wolf Labs traced multiple June 2026 intrusions to the Qilin (Agenda) gang abusing a high-severity PAN-OS authentication bypass as an entry point ahead of ransomware deployment. ([source](https://thehackernews.com/2026/07/qilin-ransomware-attackers-exploit-pan.html))

### 5. JADEPUFFER Deploys ENCFORGE Ransomware Against Langflow AI Infrastructure
**Why it matters for audit/CISO/VAPT:** Ransomware operators are now targeting model weights and vector databases directly — AI/ML pipelines need the same backup and access-control scrutiny as other crown-jewel data.
Sysdig linked a second attack on the same Langflow server (see item 2, CVE-2026-0770) to the same AI-agent-driven operator, now deploying a Go-based ransomware built to encrypt AI model files. ([source](https://thehackernews.com/2026/07/new-encforge-ransomware-targets-ai.html))

### 6. Zimbra Patches Critical SNMP Command Injection and Four XSS Bugs
**Why it matters for audit/CISO/VAPT:** Unauthenticated command injection in a monitoring component is a reminder to scope vulnerability scanning to management interfaces, not just user-facing services.
Zimbra 10.1.20 fixes nine issues, topped by an SNMP-based command injection flaw, across the widely deployed collaboration platform. ([source](https://thehackernews.com/2026/07/zimbra-patches-critical-snmp-command.html))

### 7. ServiceNow AI Platform Flaw Exploited for Unauthenticated Code Execution
**Why it matters for audit/CISO/VAPT:** A sandbox-escape RCE (CVSS 9.5) in a widely used GRC/ITSM platform is a direct risk to systems many audit and compliance workflows depend on.
Defused Cyber reports in-the-wild exploitation of CVE-2026-6875, a sandbox escape in the ServiceNow AI Platform. ([source](https://thehackernews.com/2026/07/critical-servicenow-ai-platform-flaw.html))

### 8. Bit2Watt: GPU-Based Cloud Attack Could Disrupt Power Grids
**Why it matters for audit/CISO/VAPT:** A novel attack needing no exploit or breach — just ordinary GPU workload manipulation — expands the threat model VAPT scopes should cover for cloud-hosted critical infrastructure.
Zhejiang University researchers describe how a cloud tenant can swing a data center's power draw fast enough to stress the grid it runs on, using only standard GPU access. ([source](https://thehackernews.com/2026/07/new-bit2watt-attack-could-let-cloud.html))
