# Security & Audit Digest — 2026-07-14

## Top items

### 1. Joomla iCagenda & Balbooa Forms flaws added to CISA KEV — active zero-day exploitation
**Why it matters for audit/CISO/VAPT:** Two maximum-severity RCE flaws are being exploited in the wild via arbitrary file upload; any Joomla footprint needs immediate patching and KEV-driven remediation tracking.
CISA added both extensions to its Known Exploited Vulnerabilities catalog on 2026-07-13 following confirmed zero-day exploitation. Auditors should verify plugin inventories and emergency-patch SLAs are being met. ([source](https://thehackernews.com/2026/07/icagenda-and-balbooa-forms-joomla-flaws.html))

### 2. CISA/FSB Center 16 advisory — Russian state actors exploiting poorly configured routers
**Why it matters for audit/CISO/VAPT:** Nation-state campaign against network devices across critical sectors; directly tests network-hardening controls in audit scope.
CISA and eight allied agencies (AA26-194A, 2026-07-13) warn that Russian FSB-linked actors continue to opportunistically compromise vulnerable and misconfigured routers, recommending disabling SNMPv1/v2 and Smart Install, and blocking SNMP/TFTP/SMB from the internet. ([source](https://www.cisa.gov/news-events/cybersecurity-advisories/aa26-194a))

### 3. Progress ShareFile — emergency shutdown of internet-exposed Storage Zone controllers
**Why it matters for audit/CISO/VAPT:** No patch was available at time of advisory, making this an immediate exposure-assessment and manual-mitigation action item for CISOs.
Progress Software directed customers to manually shut down internet-facing ShareFile Storage Zone controllers due to a credible external threat, as covered in SANS Stormcast's July 13 episode. Organizations running ShareFile should confirm exposure status now. (Covered in item 3; see podcast briefings below.) ([source](https://isc.sans.edu/podcastdetail/10004))

### 4. Jscrambler npm package backdoored with cross-platform infostealer
**Why it matters for audit/CISO/VAPT:** Supply-chain compromise of a widely used build-time dependency; audit should confirm no affected version reached production or CI/CD pipelines.
The jscrambler 8.14.0 npm release (published 2026-07-11) carried a preinstall hook dropping a Rust infostealer with Windows, macOS, and Linux binaries, downloaded roughly 1,500 times before removal. ([source](https://thehackernews.com/2026/07/compromised-jscrambler-8140-npm-release.html))

### 5. Critical Zimbra Classic Web Client stored XSS enables code execution
**Why it matters for audit/CISO/VAPT:** Email is a core perimeter control; an XSS-to-RCE chain triggered by a crafted email is a high-impact, low-user-interaction attack path.
Zimbra is urging customers to patch a critical stored cross-site scripting flaw in the Classic Web Client that can let a crafted email execute arbitrary code in a user's session. ([source](https://thehackernews.com/2026/07/critical-zimbra-flaw-could-let-crafted_0483473395.html))

### 6. Forg365 phishing-as-a-service targets Microsoft 365 via device code + AitM
**Why it matters for audit/CISO/VAPT:** Combines MFA-bypass techniques with AI-assisted lures; audit teams should review Entra ID sign-in logs and conditional access policies for device-code anomalies.
A new PhaaS operation, Forg365, chains device code phishing, adversary-in-the-middle session theft, antibot evasion, and AI-generated lures to compromise M365 mailboxes, with observed post-compromise mailbox abuse. ([source](https://thehackernews.com/2026/07/forg365-phaas-targets-microsoft-365.html))

### 7. CrashStealer macOS malware bypasses Gatekeeper via notarized dropper
**Why it matters for audit/CISO/VAPT:** Defeats a core macOS trust control (notarization), undermining assumptions in endpoint security baselines.
CrashStealer poses as Apple's crash-reporting tool and uses a notarized dropper to pass Gatekeeper checks, then harvests credentials, keychain data, and crypto wallets. ([source](https://thehackernews.com/2026/07/crashstealer-macos-malware-uses.html))
